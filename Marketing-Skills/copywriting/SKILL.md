# Copywriting Skill

## Before You Write Anything
1. Read .claude/rules/psychology.md — apply at least 3 principles per piece.
2. Read .claude/rules/tone.md — every sentence passes through this filter.
3. Identify which psychological trigger leads the piece (incongruity, hope, scapegoat, etc.)

## Copy Structure Default
- Open with incongruity or pattern interrupt — never a feature, never a tagline.
- Establish the problem using the prospect's exact language.
- Name the villain (the system, not the business owner).
- Introduce Social Patter as the logical conclusion, not the pitch.
- Close with a single, low-friction next step.

## Output Format
- Provide a headline, subheadline, and body unless instructed otherwise.
- Flag which psychology principles were applied and where.
- Offer one alternate headline using a different psychological angle.
