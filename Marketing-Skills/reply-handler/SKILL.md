# Skill: Reply Handler

## Changelog
- 2026-04-12 v1: Initial version -- multi-step process with table-based classification
- 2026-04-12 v2: Simplified to numbered process, added output format, added Wrong Person classification, tightened rules

## Trigger
Use this when a prospect has replied to cold outreach and a response is needed.

## Process
1. Classify the reply: Interested / Objection / Not interested / Wrong person
2. If Interested: draft a short reply that confirms interest and pushes to book a call
3. If Objection: identify the objection type (price / trust / timing / relevance) and respond using the relevant reframe in OFFERS.md
4. If Not interested: log and close the lead
5. If Wrong person: ask for the right contact

## Output Format
- Classification:
- Suggested Reply:
- Next Action in LEADS.md:

## Rules
- Keep replies under 5 sentences
- Never pitch again in the reply -- just advance to the call
- Use the person's name and reference their specific business if known
- Reply within the same channel (email reply = email, DM reply = DM)
- Match the prospect's length and energy level
- Never pitch harder when someone pushes back -- pull back and reframe

## Response Templates by Classification

### Interested
"[Name] -- great. What does your calendar look like this week for a 15-minute call? I can show you exactly how it works for a [vertical] practice your size."

### Objection: price
Use reframe from OFFERS.md. Lead with the cost of inaction, not the cost of the service.

### Objection: timing
"That's usually when this helps most. A system that runs itself doesn't add to your workload -- it takes things off it. I'll check back in [30 days]."

### Objection: trust / "we tried this"
"Fair. Most of what's out there was built for tech companies. This is built specifically for [vertical]. Happy to show you the difference in 15 minutes -- no commitment."

### Objection: relevance / "we have a receptionist"
"This isn't meant to replace her. It's for the hours she's not there. Most practices miss 3 to 5 calls per week after 5pm. Worth seeing if that's happening at yours."

### Not interested
"Understood -- removing you now. If anything changes, you know where to find me."

### Wrong person
"Thanks for the heads up -- who would be the right person to send this to?"
